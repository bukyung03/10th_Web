- 기타 태그 추가 정리해보기 🍠
    
    **폼 관련 태그들:**
    
    - `<form>` : 사용자 입력 양식
    - `<input>` : 다양한 입력 필드
    - `<button>` : 클릭 가능한 버튼
    - `<select>`, `<option>` : 드롭다운 선택 메뉴
    
    **텍스트 관련 태그들:**
    
    - `<strong>` : 중요한 텍스트 (굵게)
    - `<em>` : 강조 텍스트 (기울임)
    - `<mark>` : 하이라이트된 텍스트
    
    ---
    
    **추가로 태그를 정리해주세요:**
    
    위에서 알려주신 주요 태그들입니다.
    
    **주요 시맨틱 태그들:**
    
    - `<header>` : 웹사이트나 섹션의 머리말 부분
    - `<nav>` : 네비게이션(메뉴) 링크들
    - `<main>` : 문서의 주요 내용
    - `<article>` : 독립적인 글이나 콘텐츠 (블로그 포스트, 뉴스 기사 등)
    - `<section>` : 문서의 구획을 나누는 영역
    - `<aside>` : 본문과 관련 있지만 독립적인 콘텐츠 (사이드바, 광고 등)
    - `<footer>` : 웹사이트나 섹션의 바닥글
    
    **메타 정보 관련 태그들:**
    
    - `<head>` : 문서의 메타 데이터 영역
    - `<title>` : 웹페이지 제목
    - `<meta>` : 문자 인코딩. 설명 등 메타 정보
    - `<link>` : 외부 CSS 연결
    - `<style>` : 내부 CSS 정의
    
    **목록 관련 태그들:**
    
    - `<ul>` : 순서 없는 리스트
    - `<ol>` :  순서 있는 리스트
    - `<li>` : 리스트 항목
    
    **텍스트 관련 태그들:**
    
    - `<h1>` ~ `<h6>` : 제목, 숫자가 작을수록 글씨 사이즈가 작아짐
    - `<p>` : 문단
    - `<br>`  : 줄 바꿈
    - `<hr>`  : 구분 선
    - `<blockquote>`  : 긴 인용문을 표시 할 때 사용, 들여쓰기 형식으로 표현됨
    - `<q>`  : 짧은 인용문에 사용함, 따옴표가 자동으로 붙음
    
    **인 라인 텍스트 관련 태그들:**
    
    - `<span>`  : 인라인 영역 구분
    - `<strong>` : 중요한 텍스트
    - `<em>` : 강조 텍스트
    - `<small>` : 작은 텍스트
    - `<sub>` / `<sup>` : 아래 / 위 첨자
    
    **링크, 미디어 관련 태그들:**
    
    - `<a>` : 하이퍼링크 생성
    - `<img>` : 이미지 삽입
    - `<video>` : 동영상 삽입
    - `<audio>` : 오디오 삽입
    
    **테이블 관련 태그들:**
    
    - `<table>` : 표 전체
    - `<tr>` : 행(row)
    - `<td>` : 데이터 셀
    - `<th>` : 제목 셀
    - `<thead>` / `<tbody>` / `<tfoot>` : 구조 구분
    - `<caption>` : 표 제목
    
    **기타  태그들:**
    
    - `<div>` : 영역 구분 (레이아웃용)
